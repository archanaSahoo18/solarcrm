# Solar CRM ☀️

A CRM system to manage solar installation customers, contracts, and pipeline.

## 🚀 Features
- Customer management
- Contract upload
- Pipeline tracking
- Role-based access

## 🛠 Tech Stack
- Frontend: React
- Backend: Spring Boot
- Database: MySQL

## ▶️ Run Locally

### Backend
cd backend
mvn spring-boot:run

### Frontend
cd frontend
npm install
npm start
